POLARIS

Client

A lightweight Bedrock utility client packed with HUD modules, visual enhancements, and quality-of-life tools — all toggled through an in-game click GUI.

Download GitHub

What's inside

Built for Bedrock

Everything you need in one clean DLL — no bloat, no launchers you don't control.

◎

HUD Overlays

CPS counter, coordinates, ping display, armor status, keystrokes, combo counter, and more — all rendered directly on-screen via DX12.

✦

Visual Modules

Fullbright, hitboxes, block outlines, low fire, no hurt cam, and zoom — see the game the way you want to.

⚙

Utility Tools

Fast inventory, drop inventory, VSync disabler, better tooltips — small tweaks that make a big difference in gameplay.

⚙

Click GUI

Toggle and configure every module in-game with a clean ImGui-based click GUI. Open with L, close with Escape.

►

Per-Module Keybinds

Assign any key to any module and toggle features on the fly without ever opening the GUI.

◆

Clean Launcher

A dark-themed standalone launcher auto-detects Minecraft and injects with one click — no command line required.

Get started

Download & Inject

Download the latest build from GitHub Actions, drop the files together, and inject with one click.

01

Download

Go to the GitHub Actions page and download the latest PolarisClient artifact.

Windows x64
02

Install Runtime

Install the Visual C++ Redistributable 2015–2022 x64 if you haven't already.

One-time setup
03

Open Minecraft

Launch Minecraft for Windows from the Start Menu and wait until you're in the game.

Bedrock Edition
04

Inject

Run PolarisLauncher.exe as Administrator and click Inject into Minecraft. Press L in-game to open the GUI.
